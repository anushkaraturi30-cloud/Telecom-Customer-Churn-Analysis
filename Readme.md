# 📊 Telecom Customer Churn Analysis (Excel-Based Business Intelligence Project)

---

# 🚀 Project Overview

This project presents a structured end-to-end Customer Churn Analysis conducted using Microsoft Excel on a real-world telecom dataset containing 7,043 customer records.

The objective of this analysis was to identify key churn drivers, quantify business impact, and develop actionable retention strategies. The project combines data cleaning, analytical modeling, KPI development, segmentation logic, scenario analysis, and interactive dashboard reporting.

This work demonstrates practical business-focused data analysis using Excel as a structured analytical tool.

---

# 🎯 Business Problem

Customer churn directly affects revenue stability and long-term growth in subscription-based industries such as telecom.

The organization required clarity on:

- Which customers are most likely to churn
- What factors influence churn behavior
- How churn risk can be reduced strategically
- What revenue impact churn reduction could generate

This project addresses these questions through structured data exploration and quantitative analysis.

# 🛠 Analytical Approach

The project was developed in multiple structured stages:

## 1️⃣ Data Cleaning & Preparation

- Removed duplicates  
- Converted text to structured columns  
- Standardized numerical fields  
- Applied data validation checks  
- Corrected formatting inconsistencies  

## 2️⃣ Lookup & Data Retrieval Modeling

- Implemented XLOOKUP for dynamic customer data retrieval  
- Applied INDEX & MATCH for advanced lookup functionality  
- Built a dedicated Lookup_Model sheet to demonstrate modern Excel capabilities  

## 3️⃣ KPI & Conditional Metrics

Developed business-focused KPIs using COUNTIFS and SUMIFS:

- High Risk Customer Count  
- Revenue Lost Due to Churn  
- Segment-based churn metrics  

These metrics support quantitative churn evaluation.

## 4️⃣ Customer Risk Segmentation Model

Developed a structured Risk_Category classification based on:

- Contract Type  
- Customer Tenure  
- Business-defined churn logic  

Segments created:

- High Risk    
- Normal Risk  

This segmentation supports targeted retention strategy development.

## 5️⃣ Advanced Filtering & Business Logic Extraction

Implemented Advanced Filter to extract high-risk customer segments using multi-criteria conditions:

- Contract = Month-to-Month  
- Churn = Yes  
- Tenure < 6 Months  

This demonstrates structured data extraction beyond basic filtering.

## 6️⃣ Scenario & What-If Analysis

Applied Goal Seek to simulate churn reduction scenarios.

Business Question Modeled:
"What is the revenue impact if churn rate reduces by 5%?"

This extends the analysis from descriptive reporting to strategic planning.

## 7️⃣ Dashboard & Visualization

Built an interactive executive-level dashboard featuring:

- Total Customers KPI  
- Total Churn KPI  
- Overall Churn Rate  
- Risk Segment Distribution  
- Churn by Contract Type  
- Churn by Tenure Group  
- Conditional formatting for churn risk thresholds  
- Dynamic filtering using slicers  

The dashboard is designed for clarity, executive interpretation, and business decision support.

---

# 📊 Key Analytical Insights

- Month-to-Month contract customers show significantly higher churn rates.  
- Customers within their first 6 months represent the highest churn risk segment.  
- Higher monthly charges correlate with increased churn probability.  
- Long-term contract holders demonstrate stronger retention stability.  
- Risk segmentation highlights a concentrated pool of high-risk customers requiring targeted intervention.  

---

# 💡 Strategic Recommendations

- Promote long-term contract conversion for Month-to-Month customers  
- Strengthen onboarding programs during first 6 months  
- Implement targeted retention campaigns for high-risk segment  
- Evaluate pricing sensitivity among churn-prone customers  
- Monitor churn KPIs continuously using structured dashboard reporting
